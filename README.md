### Technologies Used

* HTML
* Bootstrap 5

---

## 2️ Palindrome Checker

This program checks whether the entered string is a **palindrome** or not.

A palindrome is a word that reads the same **forward and backward**.

### Example

```
madam → Palindrome  
hello → Not Palindrome
```

### Logic Used

* Reverse the string using:

```
split("").reverse().join("")
```

* Compare the original and reversed string.
